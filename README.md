## Getting Started

To get a local copy up and running follow these simple steps.

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/chungyau97/mvc-tutorial-1.git
   ```
2. Build the project
3. Start Debugging by pressing (F5)
